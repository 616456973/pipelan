# RAS CRM

A single-file HTML Web App for managing RAS CRM opportunity/sales data. The .xlsx file is the database.

See spec for design.

## Quick Start

1. Double-click `ras_crm.html`
2. Click "Open" and select your xlsx file
3. Edit / add / delete records
4. Click "Save" -- a new xlsx is downloaded; copy it back over your original
